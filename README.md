# Sistema Maçônico

Sistema completo para gestão de Lojas Maçônicas, incluindo:

- Gestão de obreiros com histórico de graus
- Gestão de reuniões e atas
- Sistema de sindicância para candidatos
- Gestão de familiares e condecorações
- Controle de presença
- Sistema de permissões por grau
- Backup e restauração
- Notificações por e-mail

## Tecnologias

- Python 3.12+
- Flask 2.3.3
- PostgreSQL
- Bootstrap 5
- ReportLab (PDF)
- OpenPyXL (Excel)
